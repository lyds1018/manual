## 🛠 uv 配置克隆项目环境

### 1. 进入项目目录

```shell
cd my_project
```

------

### 2. 创建虚拟环境

```python
uv venv
uv venv --python 3.10
```

------

### 3. 激活虚拟环境

- **Linux / macOS**:

  ```shell
  source .venv/bin/activate
  ```

- **Windows (PowerShell)**:

  ```shell
  .venv\Scripts\activate
  ```

------

### 4. 安装依赖

- 如果项目有 **requirements.txt**：

  ```shell
  uv pip install -r requirements.txt
  ```

- 如果项目有 **pyproject.toml**：

  ```shell
  uv sync
  ```

  （会自动解析 `pyproject.toml` 和 `uv.lock`，并安装依赖）
