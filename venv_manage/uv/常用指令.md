# 🐍 uv 常用指令

## 1. 项目初始化
```bash
uv init myproject           # 初始化一个新项目
uv python install 3.11      # 安装指定 Python 版本
```

## 2. 依赖管理
```bash
uv add requests             # 添加依赖
uv add "numpy==1.26.0"      # 添加指定版本依赖
uv add --dev black          # 添加开发依赖

uv remove requests          # 移除依赖
uv sync                     # 按 lock 文件同步依赖
uv lock                     # 生成或更新 lock 文件

uv update requests          # 更新指定依赖
uv update --all             # 更新所有依赖

uv list                     # 列出依赖
uv tree                     # 查看依赖树
uv check                    # 检查兼容性
```

## 3. 环境管理
```bash
uv python list              # 列出可用 Python 版本
uv python install 3.10      # 安装指定 Python 版本
uv python pin 3.11          # 固定项目 Python 版本
```

## 4. 运行程序
```bash
uv run python main.py       # 运行 Python 文件
uv run pytest               # 运行测试
uv run jupyter notebook     # 启动 Jupyter
```

## 5. 发布
```bash
uv build                    # 构建包
uv publish                  # 发布到 PyPI
```
